ibis.expr.api.TableExpr.semi_join¶

TableExpr.semi_join(other, predicates=())¶

Perform a relational join between two tables. Does not resolve resulting table schema.

Parameters:

left : TableExpr

right : TableExpr

predicates : join expression(s)

how : string, default ‘inner’

  • ‘inner’: inner join
  • ‘left’: left join
  • ‘outer’: full outer join
  • ‘right’: right outer join
  • ‘semi’ or ‘left_semi’: left semi join
  • ‘anti’: anti join
Returns:

joined : TableExpr

Note, schema is not materialized yet

ibis.expr.api.GroupedTableExpr.size¶

GroupedTableExpr.size(metric_name='count')¶

Convenience function for computing the group sizes (number of rows per group) given a grouped table.

Parameters

metric_name (string, default 'count') – Name to use for the row count metric

Returns

aggregated – The aggregated table

Return type

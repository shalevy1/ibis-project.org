ibis.omniscidb.api.OmniSciDBClient.database¶

OmniSciDBClient.database(name=None)¶

Connect to a given database.

Parameters

name (str, optional) – The name of the database to connect to. If None, return the database named self.current_database.

Returns

db – An ibis.client.Database instance.

Return type

Database

Notes

This creates a new connection if name is both not None and not equal to the current database.

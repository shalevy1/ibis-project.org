ibis.impala.api.ImpalaTable.partitions¶

ImpalaTable.partitions()¶

Return a pandas.DataFrame giving information about this table’s partitions. Raises an exception if the table is not partitioned.

Returns

partitions

Return type

pandas.DataFrame
